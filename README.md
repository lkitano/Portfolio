# Portfolio

Welcome to my portfolio. Here are some projects I have worked on, both in person and others, over the years. Feel free to check out the code and the datasets I used in each analysis. This portfolio contains the following projects:

## Penguins
This is a project that seeks to explore and visualize the Palmer Penguins dataset. This is a toy dataset I used to demonstrate regression analysis, machine learning classification, and data visualization.

## Cars
This is a project highlighting my skills in data analysis and machine learning classification. This project explores the Car Quality dataset and how to make an ML classifier with a dataset of many catregorical variables.

## Twitter
This is a collabortive project I worked on to analyze Twitter Tweets about the LGBT+ community and how it correlates with hate crimes towards this community across the United States. This project included the use of Natural Language Processing to analyze the sentiment of each tweet from a dataset of hundreds of thousands of observations.
