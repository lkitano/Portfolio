This project is large enough to get its own README file! This was a collaborative project I worked on to analyze the sentiment of tweets regarding the LGBT+ community on twitter and how it correlates to hate crimes toward the LGBT across the United States. The original Twitter Dataset cannot be shared due to logistic reasons and agreements made with UC Irvine, so a much smaller dataset is provided below in tweets_num_crimes.csv (Be warned - some of the tweets are wild: read at your own risk). The hate crimes dataset can be downloaded at https://cde.ucr.cjis.gov/LATEST/webapp/#/pages/downloads#datasets.

A large portion of this project inlcuded Sentiment Analysis, or the process of scoring how "positive" or "negative" a string of text is. Sentiment Analysis is a complex process of ML classification because it involves editing strings of text through Stemming, Lemmatization, and Tokenization. Furthermore, the actual results of Sentiment Analysis, classififying a string of text to be "positive" or "negative", can be up to subjective opinion. Nevertheless, we as a group found our ML Classifier to be a satisfactory model for our Twiitter Dataset. For a brief summary of our findings, check out my summary at Findings.pdf.
